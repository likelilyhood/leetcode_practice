## 常见算法技巧
- 动态规划建模：先定义状态，再写转移，典型有线性 DP、二维 DP、分组 DP、滚动数组优化。
- 逆向 DP 和倒序遍历：当后面的状态更容易确定时，从后往前推，常用于带选择和剩余容量的题。
- 分组背包思想：每一组只能选一个或选若干个候选项时，把“组”作为转移单位。
- 浮点转整数：把损耗、比例、金额类数据放大成整数，减少精度误差。
- 聚类迭代：先分配样本，再更新中心点，循环直到中心不变。
- 距离函数替换：欧氏距离、IOU、加权距离都可以替换，关键是先定义什么叫相似。
- 边界判断和不可达处理：用 INF、-1、提前判无解等方式，避免状态污染。
